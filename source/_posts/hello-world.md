## Welcome to My Blog

This is my first blog post! I built this blog using **Hexo** and the **Butterfly** theme, hosted on **GitHub Pages**.

### Why Hexo?

Hexo is a fast, simple, and powerful Node.js-based static blog framework. Key features:

- **Markdown support** - Write posts in Markdown
- **Fast generation** - Generate hundreds of pages in seconds
- **Rich plugin ecosystem** - Extend functionality easily
- **One-command deployment** - Deploy to GitHub Pages with a single command
